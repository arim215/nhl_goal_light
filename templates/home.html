NHL GOAL LIGHT !!

API DOCUMENTATION

/api/v1/season Return true if a Hockey season is in progress

/api/v1/teams : Get the list of teams

/api/v1/team/<team>/id : Get the ID of a team, ex: /api/v1/team/Canadiens/id

/api/v1/team/<id>/game : Return true if a game is in progress for the given team ID. Ex: /api/v1/team/8/game

/api/v1/team/<id>/score : Get the score of a given team ID, ex: /api/v1/team/8/score

/api/v1/team/<id>/end_game : Return TRUE if game is over for the given team ID. Ex: /api/v1/team/8/end_game

/api/v1/goal_light/activate : Activate the Goal light
